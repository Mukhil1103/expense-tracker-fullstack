# 💰 Expense Tracker (Full Stack)

A full-stack expense tracking application built with:

- ⚛️ React (Frontend)
- 🐍 Django REST Framework (Backend)
- 🐘 PostgreSQL (Database)
- ☁️ Vercel + Render Deployment

## 🚀 Features
- JWT Authentication
- Add / Edit / Delete expenses
- Dashboard analytics (charts + stats)
- Secure API
- Responsive UI
